version 0.1可以实现在padding上面添加perturb来进行攻击
version 0.2 可以实现直接用perturb替换为对应的padding来进行攻击,并且添加了constraint
由于本身设置的步长过长所以说单独设置了保存uap
version 0.3 添加了一个尽可能靠近10个最近embedding的loss来保证尽可能的学习到相关信息.
version 0.4 并非每一次都直接进行clip,而是优化一段时间进行clip,效果很糟糕.
version 0.5 现在已经优化到了,会保存loss值最小的embedding来进行学习,而非之前的保存最后一个,但是可能还是有问题
version 0.6 优化到了最后可以选取相似度前10的tensor来计算
version 0.7 虽然是每次都会是优化一段时间之后进行clip,但是通过把限制embedding的loss尽可能constriant的前提下,发现可以实现较高的攻击率
version 0.8 通过把embedding attack本身中的generate 进行限制的基础之上实现了保证复现完美. 这个版本目前可以生成完全可复现的token还有对应的攻击以及翻译
version 0.9 就是尝试了不同的参数来进行限制最后得出最优参数
目前的embedding_attack的结果基本上算是可以满意了,开始进行uap的优化
